# 🛒 DUNKER'S E-Commerce Website

A full-featured e-commerce web application with a personalized product recommendation system built as my final project.

## 🚀 Features
- Product catalog with search and filter
- Personalized product recommendations using Collaborative Filtering
- Shopping cart and transaction management
- Responsive design for desktop and mobile
- Admin dashboard for product and order management

## 🛠️ Tech Stack
- **Backend:** PHP, Laravel
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **Design:** Figma
